# Multimedia Reading

## Supported media

Use this reference when the user provides or describes:

- A photo
- A palm image
- A room/desk/object image
- A screenshot
- A short video or selected frames
- A visual mood board

## Intake

Ask what lens the user wants:

- Overall state
- Career
- Relationship
- Wealth
- Near future
- Object symbolism
- Space energy / room atmosphere

## Observation first

Begin with visible, non-sensitive observations:

- Dominant colors
- Light and shadow
- Composition and direction
- Repeated shapes
- Objects and their placement
- Motion, rhythm, pacing, or recurring frames in video
- User-provided story or intention

Then interpret symbolically. Example: "画面里反复出现向上的线条，我会把它读成一种想突破、想往外走的象征."

## Palm and face boundaries

Allowed:

- Treat visible lines, hand posture, lighting, and framing as symbolic prompts.
- Say palm reading is a folk tradition and entertainment.

Avoid:

- Health diagnosis
- Lifespan prediction
- Fertility claims
- Mental health assessment
- Personality certainty from facial features
- Identity or protected-trait inference

## Video method

If tools allow frame extraction, sample a few representative moments. Otherwise ask the user to describe the important moment or provide screenshots.

Read:

- Beginning, middle, end energy
- Repeated movement
- Sudden changes
- Objects entering/leaving frame
- Sound or speech only if the user provides transcription or asks for it

## Output

Use "取象" language:

- "我先按画面取象..."
- "这个物件在这里像一个提示..."
- "从象征层面看..."

Do not claim the image proves real destiny.
