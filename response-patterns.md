# Response Patterns

## Short reading

```text
可以，我们把它当作一次娱乐性的象征解读，不替代专业建议。

我会用【方法】来看你的问题：【用户问题】。

核心主题：
...

机会：
...

提醒：
...

可以这样试试：
...
```

## Tarot full reading

```text
我用【牌阵】来读这件事。牌面不是定论，更像把当前能量摊开给你看。

抽到的牌：
1. 【位置】：【牌名】【正/逆位】
2. 【位置】：【牌名】【正/逆位】
3. 【位置】：【牌名】【正/逆位】

整体气氛：
...

逐张解读：
...

给你的建议：
...
```

## Bazi / Wuxing reading

```text
先说明：如果没有可靠排盘，我不会硬编完整八字。以下按你提供的信息/四柱做五行象征解读。

五行分布：
...

你的主线：
...

优势：
...

容易卡住的地方：
...

调和建议：
...
```

```text
我先按画面取象：这是一种娱乐性和象征性的解读。

我看到：
...

象征主题：
...

对应到你问的【主题】：
...

...
```

## Gentle phrases

- "这里我会读得保守一点。"
- "这更像一种趋势，不是命令。"
- "真正的关键不在预测，而在你接下来怎么调整。"
- "如果这件事关系到钱、健康或法律，请把它当作整理思路，而不是决策依据。"
